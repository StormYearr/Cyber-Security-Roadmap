## üõ†Ô∏è Cyber Security Roles Overview

### 1. Security Analyst (Defensive)
* **Focus:** Monitoring and maintaining security controls.
* **Responsibilities:** Analyzing networks to uncover actionable data, compiling safety reports, and developing security plans.

### 2. Security Engineer (Defensive/Infrastructure)
* **Focus:** Building and implementing security solutions.
* **Responsibilities:** Testing security measures, monitoring networks, and updating systems to mitigate vulnerabilities.

### 3. Incident Responder (Defensive/Reactive)
* **Focus:** Rapid response to active breaches.
* **Key Metrics:** MTTD (Detect), MTTA (Acknowledge), and MTTR (Recover).
* **Responsibilities:** Developing IR plans and reducing damage during unfolding attacks.

### 4. Digital Forensics Examiner (Investigative)
* **Focus:** Playing "Detective" to solve digital crimes.
* **Responsibilities:** Collecting evidence following legal procedures and analyzing file systems/memory to identify perpetrators.

### 5. Malware Analysis (Deep Technical)
* **Focus:** Reverse-engineering suspicious programs.
* **Responsibilities:** Static and dynamic analysis of malware to understand its behavior and create detection signatures.

### 6. Penetration Tester (Offensive)
* **Focus:** Systematic hacking to find flaws (Legal/Safe environment).
* **Responsibilities:** Testing web apps and networks to evaluate risk and recommend patches.

### 7. Red Teamer (Advanced Offensive)
* **Focus:** Emulating real-world adversaries to test detection/response capabilities.
* **Responsibilities:** Long-term engagements aimed at avoiding detection and testing a company's "Blue Team" readiness.
